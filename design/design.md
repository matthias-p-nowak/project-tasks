# Description

This is a project management support tool. The main page shows a hierarchical list of tasks together with their status, optional due dates, list of assigned people and a priority.

# Folder outline

- When deployed, the folder `public` will be accessible from the internet.
- The folder `private` is accessible by the *PHP* engine and the `main.php` therein is imported into the `index.php` via a `require` statement.
